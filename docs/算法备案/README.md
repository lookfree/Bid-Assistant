## 备案口径速记(与实现一致的关键事实)

1. **算法类型**:生成合成类为主(标书文本生成),辅以检索(RAG)与审核辅助(风险清单,人工复核)。**无推荐、无排序、无用户画像**——用户权益里"关闭推荐/删除画像"不适用。
2. **模型全部外采推理或开源自部署,零训练**:不使用用户数据训练是硬事实,可直接承诺。
3. **每一步用户显式触发**、失败自动退款、结果可编辑可溯源(clause_id 引用)——用户可控性强,是备案陈述的加分项。
4. **应急开关真实存在**:运营后台移除模型配置 → 服务端 400 拒绝生成,等效一键停用算法。
5. Mermaid 图导出:可用 IDE 插件或 mermaid.live 渲染后截图/导出 PNG 附入备案系统。
