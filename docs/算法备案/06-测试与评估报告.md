## 二、生产实测(2026-07,真实大型招标文件)

**测试对象**:约 1MB、9167 条款、四包件的真实招标文件(脱敏后可提供样例截图)。

| 指标 | 实测结果 |
|---|---|
| 读标分段完成率 | 95/95 轮全部成功 |
| 故障续跑复用率 | 中断重试时 94/95 轮直接复用缓存(不重复消耗) |
| 技术需求提取量 | 2628 条结构化技术条目,包件划分 4 包正确 |
| 单项目向量检索延迟 | 292ms(2s 超时上限内) |
| 异常处理 | 期间 8 次生成失败,**全部自动全额退款,用户净损失 0**(账本可查) |
| 长任务稳定性 | 晚高峰 ~11 分钟单轮在 1200s 时长盖内正常完成 |
| 读标结果可溯源 | 每条解读带原文条款引用(clause_id),抽查与原文一致 |

**评分办法空结果核对案例**:某项目评分办法(scoring)输出为空,人工回查原文确认该标采用"经评审的最低投标价法"(无打分表),空结果**正确**——证明算法不虚构不存在的内容。
