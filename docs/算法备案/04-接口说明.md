# 接口说明

> 仅列与算法调用直接相关的接口。全部接口经 Nginx HTTPS 暴露,除登录外均需会话鉴权;智能体服务接口不对公网暴露。

## C 端 App API(对外)

### 认证

| 接口 | 方法 | 说明 |
|---|---|---|
| `/auth/sms/send` | POST | 发送登录验证码(冷却+尝试次数限制) |
| `/auth/sms/verify` | POST | 验证码登录,签发会话(7 天) |
| `/auth/me` | GET | 当前用户信息 |

### 项目与算法步骤(核心)

| 接口 | 方法 | 说明 |
|---|---|---|
| `/projects` | POST | 创建项目(绑定已上传的招标文件 key) |
| `/projects/:id?slim=1` | GET | 项目状态(slim 模式仅返回步骤状态列,不含大结果) |
| `/projects/:id/steps/read` | POST | 触发**智能读标**(显式点击;预扣积分,失败退还) |
| `/projects/:id/steps/outline` | POST | 触发**提纲生成** |
| `/projects/:id/steps/content` | POST | 触发**正文生成** |
| `/projects/:id/steps/review` | POST | 触发**标书审查** |
| `/projects/:id/steps/present` | POST | 触发**述标演示生成** |
| `/projects/:id/steps/export` | POST | 触发**导出渲染** |
| `/projects/:id/steps/:step/result` | GET | 按需拉取单步结果 |
| `/projects/:id/steps/:step/events` | GET(SSE) | 实时进度事件流(阶段/章节/心跳) |
| `/projects/:id/chapters/:no/rewrite` | POST | 单章重写(用户指令驱动) |

**统一前置校验**:步骤顺序校验(409)、模型未配置拒绝(400 `model_not_configured`,不占步位不预扣)、幂等防重复启动。

### 文件与资料库

| 接口 | 方法 | 说明 |
|---|---|---|
| `/files/presign` | POST | 上传预签名(直传 MinIO) |
| `/library` | GET/POST/DELETE | 用户资料库 CRUD(变更后 best-effort 通知智能体索引/删索引) |

## 智能体服务 API(仅内网,App API 调用)

| 接口 | 方法 | 说明 |
|---|---|---|
| `/agents/bidding_agent/runs` | POST | 创建生成任务(run);携带 thread_id、step、user_id、模型编排(主+降级链,来自运营后台配置) |
| `/agents/bidding_agent/runs/:id/stream` | GET(SSE) | 任务进度流(App API 转发给前端) |
| `/rag/index` | POST | 索引文档(切块→嵌入→入库);嵌入服务不可用时返回 `{disabled:true}` 降级 |
| `/rag/index/{source_type}/{source_id}` | DELETE | 按属主(user_id 校验)删除索引 |
| `/health` | GET | 探活 |

**运行契约**:一个项目一个 `thread_id`,一步一个 `run_id`;智能体仅回报 token 用量,不接触资金;`user_id` 仅作数据隔离键。

## 运营后台 API(独立鉴权,8 小时会话)

| 接口 | 说明 |
|---|---|
| 模型编排配置 | 配置各步骤主模型+降级链、每模型思考开关(默认关);**这是模型型号的唯一来源** |
| 计费配置 | 各步骤积分价格 |
| 用户/订单/账本查询 | 运营审计 |
| 概览看板 | 用量趋势统计 |
