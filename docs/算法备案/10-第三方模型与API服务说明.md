# 第三方模型与 API 服务说明

## 一、大语言模型 API(算法核心依赖)

| 项 | 说明 |
|---|---|
| 供应商 | DeepSeek(深度求索)、智谱 AI(GLM)、阿里云(通义千问)——均为国内供应商,各自已完成生成式 AI 服务备案 |
| 实际启用型号 | 由运营后台「模型编排」配置(主模型+降级链);备案时可导出当前配置截图作为附件 |
| 调用方式 | OpenAI 兼容 HTTPS API,服务端直连(不经中间代理商) |
| 传输数据 | 招标条款文本片段、生成指令(prompt);**不传输**用户手机号、账号、资金信息 |
| 数据用途约束 | 仅推理;按各供应商 API 服务协议,推理数据不用于训练【备案时附各供应商协议链接/截图】 |
| 计费 | 按 token 用量向供应商付费;与用户积分计费完全解耦(用户按步一口价) |
| 故障预案 | 降级链自动切换;全链失败则本步失败并自动退款;运营后台可随时更换/下线供应商 |

## 二、嵌入模型(开源自部署,非第三方服务)

| 项 | 说明 |
|---|---|
| 模型 | BGE-M3(开源许可) |
| 部署 | 自有服务器 Docker 部署,仅内网访问 |
| 数据 | 文本不离开自有环境;无第三方数据传输 |

## 三、非算法类第三方服务

| 服务 | 用途 | 传输数据 | 算法相关性 |
|---|---|---|---|
| 阿里云短信 | 登录验证码 | 手机号、验证码模板参数 | 无(仅认证) |
| 收钱吧支付 | 积分充值/会员订阅 | 订单号、金额 | 无(资金链路,与算法隔离) |
| 阿里云 RDS PostgreSQL | 数据库托管 | 业务数据存储 | 基础设施 |

## 四、依赖清单式声明

- 算法推理**唯一**的外部数据出口 = 大模型 API 的 prompt 传输(HTTPS 加密、国内供应商、仅推理)。
- 无境外模型/API 依赖;无外采黑盒算法;无第三方推荐/画像 SDK。
- 供应商可替换性:模型接入为 OpenAI 兼容协议 + 配置驱动,更换供应商无需改代码,运营后台操作即可生效。
